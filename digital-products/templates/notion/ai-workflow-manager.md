### Workflow Templates

#### Content Workflow
1. **Ideation** → Use Claude to generate 10 topic ideas
2. **Research** → Use GPT to gather key points
3. **Outline** → Create structure in Notion
4. **Draft** → Write first draft with AI assistance
5. **Polish** → Human edit and refine

#### Coding Workflow
1. **Spec** → Write detailed spec
2. **Generate** → AI generates initial code
3. **Review** → AI reviews its own output
4. **Test** → Write and run tests
5. **Refactor** → Clean and optimize

#### Research Workflow
1. **Define** → Set research questions
2. **Gather** → Use AI to find sources
3. **Synthesize** → AI summarizes findings
4. **Output** → Create final report

## QUICK REFERENCE

### Best Practices

1. **Be Specific** - Vague prompts = vague results
2. **Iterate** - First draft is just a starting point
3. **Save Winners** - Great prompts = save immediately
4. **Track Results** - Know what works
5. **Chain Workflows** - Combine AI tools for power

### Tool-Specific Tips

**ChatGPT:** Best for conversation, iteration
**Claude:** Best for long-form, analysis, coding
**Gemini:** Best for multimodal, real-time info
**Midjourney:** Best for images (use specific style codes)
